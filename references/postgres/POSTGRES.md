Getting in Psql
==============

In order to connect to PostgresSQL through the terminal:

#### change stage to root
```
 sudo -i
```

### change the user to postgres
``` 
su postgres
```

### run psql
```
psql
```

### enter the username and the password
```
ALTER USER postgres WITH PASSWORD 'postgres'
```
